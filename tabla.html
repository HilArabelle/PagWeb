Git y GitHub

Concepto Descripción Usos Imagenes Descriptivas
Git Git es un sistema de control de versiones distribuido, creado por Linus Torvalds en 2005. Permite a múltiples desarrolladores trabajar en un proyecto de manera simultánea, manteniendo un historial de cambios detallado.
  • Seguimiento de cambios en el código fuente.
  • Colaboración entre múltiples desarrolladores.
  • Reversión de código a versiones anteriores.
  • Ramas para desarrollar nuevas funcionalidades de manera aislada.
GitHub GitHub es una plataforma de desarrollo colaborativo de software basada en Git. Fue fundada en 2008 y permite a los desarrolladores alojar proyectos, colaborar en código y gestionar versiones mediante una interfaz web.
  • Alojamiento de repositorios Git.
  • Colaboración abierta en proyectos de código abierto.
  • Gestión de issues y seguimiento de errores.
  • Revisiones de código mediante pull requests.
  • Documentación de proyectos mediante wikis y archivos README.
