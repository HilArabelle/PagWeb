¿Qué es Git hub?

GitHub es un servicio basado en la nube que aloja un sistema de control de versiones (VCS) llamado Git.
Éste permite a los desarrolladores colaborar y realizar cambios en proyectos compartidos, a la vez que mantienen un seguimiento detallado de su progreso.


Caracteristicas

Guardar proyectos

Guarda proyectos de forma gratuita

Colaboracio

Los usuarios pueden entrar a los proyectos y colaborar para agregar funciones o mejorar las que ya estan disponibles

Herramientas

Tiene diversas herramientas ademas de las proporcionadas por Git

Seguimiento de problemas

Tiene un sistema de seguimineto de problemas por lo cual otras personas pueden realizar mejoras,sugerencias y optimizacion de cada proyecto

Revision de codigo

Cuenta con una herramienta de revision de codigo

Graficos

Se pueden ver graficos sobre el trabajo de lso desarrolladores y las ramidificacion de cada proyecto
